# Overview

VisionPipe is a real-time computer vision platform that enables hand tracking, gesture recognition, and 3D visualization directly in the browser.

## Features

- **Real-time Hand Tracking** - Track hand landmarks with low latency using MediaPipe
- **Multiple Effects** - Apply various visual effects to your webcam feed
- **Cloud API** - Process images and video through our REST API
- **Credit-based Billing** - Pay only for what you use
